create a new group quick and simple

groups allow you to create a collection of friends that want to play golf together

things to know

privacy
  • public: searchable and open to everyone without the need for invites or accepts
  • semi-private: searchable, but requests to join must be accepted
  • private: the only way to find or join this group is to have an invitation
names
  • length: group names must be between 5 - 50 characters
  • quality: if the group is public or semi-private, people will be able to find you based on the name
  • changes: you can change the name of a group at any time
  • description: describe your group. it'll help people find you if there are other groups with the same name
