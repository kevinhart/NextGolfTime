How It Works

Sign In

Use an existing social network account so there is no lengthy registration process.

The last thing you need is another username and password to remember.

Connect

Join up with your friends or create a group and let them join you.

Privacy controls let you decide how much others can see and who can join your events.

Read more about the features of group management.

Schedule

Quickly add events and see where you have already committed. You'll be able to easily see upcoming events for your groups.

Optionally receive email notifications of any changes.

Group Management

Creating a group allows for additional features that you won't get otherwise.

Privacy

A group can be created with one of three levels of privacy. This setting controls how much users can see of your group.

Administration

A group can only be owned by a single user but other members can be made administrators. Admins have extended privelidges that allow them to create events, accept new members, and send group emails.

Events

Group admins can create events. Events can be created with a maximum capacity if needed.

E-Mail

Group admins may send emails to all members of the group. This can be handy so you don't have to keep a separate email list.
